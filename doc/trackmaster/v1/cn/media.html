API - 媒体

获取系统媒体库列表

GET /medias

参数

sort
可选 String - 列表排序以什么排序
  • id - 按照媒体ID排序
  • name - 按照媒体名称排序
  • create_time - 按照创建日期排序
direction
可选 String - 排序方式
  • asc 升序 (默认)
  • desc 降序
page
可选 Int - 显示页码
per_page
可选 Int - 分页数量，默认每页30条

响应

Status: 200 OK
Link: <http://api.trackmaster.com.cn/medias?page=2>; rel="next",
      <http://api.trackmaster.com.cn/medias?page=10>; rel="last"
X-RateLimit-Limit: 5000
X-RateLimit-Remaining: 4999


[
  {
    "id": 1,
    "url": "http://api.trackmaster.com.cn/medias/1",
    "name": "新浪",
    "logo": "http://www.trackmaster.com.cn/data/mediaIcon/1.ico",
    "domain": "sina.com.cn",
    "tag": "综合其他",
    "created_at": "2012-09-06T20:39:23Z"
  }
]

媒体用户获取iab数据

GET /medias/:id/ies

响应

Status: 200 OK
Link: <http://api.trackmaster.com.cn/medias/1/ies?page=2>; rel="next",
      <http://api.trackmaster.com.cn/medias/1/ies?page=10>; rel="last"
X-RateLimit-Limit: 5000
X-RateLimit-Remaining: 4999

参数

pubid
可选 String - pubid 指定后只获取该pubid的数据
date
可选 Date - 日期，要查看的数据日期，YYYY-mm-dd 例如: 2012-06-08 ,不指定则获取头一天的数据
page
可选 Int - 显示页码

    "city": "北京",
    "geoid": 110000,

  }
]

获取指定工作网络下媒体库列表

GET /networks/:network_id/medias

参数

sort
可选 String - 列表排序以什么排序
  • id - 按照网络媒体ID排序
  • name - 按照网络媒体名称排序
  • status - 按照状态排序
  • framework - 按照框架排序
direction
可选 String - 排序方式
  • asc 升序 (默认)
  • desc 降序
page
可选 Int - 显示页码
per_page
可选 Int - 分页数量，默认每页30条

Status: 200 OK
Link: <http://api.trackmaster.com.cn/networks/1/medias?page=2>; rel="next",
