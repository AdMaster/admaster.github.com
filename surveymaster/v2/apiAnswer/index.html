2. 获取指定问卷的答案列表

GET /surveys/:id/answers

可选参数

  • collector_id 指定渠道id进行过滤
  • status 根据答题情况进行过滤（-1:被甄别 0:未答完 1:完成）
  • respondent_id 根据答题人进行过滤

请求

{
	"collector_id" : 1,
	"status" : 0,
	"respondent_id" : 1
}

响应

Status: 200 OK

{
	"id" : 1,/* 答案id */
	"respondent_id" : "1",
	"collector_id" : 1,
	"collector_name" : "新浪汽车",
	"created" : 123456789,// 开始答题时间
	"status" : "-1/0/1"// -1:被甄别 0:未答完 1:完成
}
