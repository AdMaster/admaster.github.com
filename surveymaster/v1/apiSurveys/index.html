获取问卷列表

GET /surveys ### 参数
per_page
可选 int - 每页显示问卷数量
  • 20（默认）
  • 10
  • 50
  • 100
sort
可选 Enum - 列表以什么方式排序
  • survey\_modify\_date - 按照修改问卷时间排序
  • survey\_create\_date - 按照创建问卷时间排序
  • survey\_titile - 按照标题排序
select
可选 String - 排序方式
  • ASC 升序
  • DESC 降序
